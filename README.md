# Python Snake Game (Turtle Version)

A classic Snake game built with Python's built-in `turtle` library.

## Features
- Arrow key or WASD controls.
- Growing snake upon eating food.
- High score tracking.
- No external dependencies required!

## Prerequisites
- Python 3.x

## How to Play
1. Run the game:
   ```bash
   python snake.py
   ```
2. Use the **Arrow Keys** or **WASD** to move the snake.
3. Eat the red food to grow and increase your score.
4. Avoid hitting the walls or your own tail!
